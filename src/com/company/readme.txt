Pour utiliser la carte, une interface est fournie pour guider l'utilisateur (classe Portail) :

***********************************************************
               GUICHET D' OPERATION DE PURSE
***********************************************************
	 1- Créditer mon purse
	 2- Débiter mon purse
	 3- Débloquer le purse
	 4- Mon solde


Si lors d'une opération de crédit, le compte se bloque, il faut revenir
au menu principal pour activer la commande permettant de la débloquer.
